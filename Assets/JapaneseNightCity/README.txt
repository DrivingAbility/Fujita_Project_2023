

--------------------
Japanese Night City
--------------------


Description:
--------------------
Japanese Night City contains the assets to make up the downtown area of Japan.

Content
--------------------
- 3 Demo scenes(Night/Day/Overview)
- 147 FBX
- 272 Prefabs
- 244 Textures (Type:Albedo ,Metalic ,Normal and Emissive Map/Size:512- 4096)
- Buildings have 2 LOD's

Triangle Count
----------------------
Prop asset: 2 - 3100
Building Prefabs: 4000 - 15000

Info
--------------------------
*Buildings have no interior.
*This package contains 3D art only, no rigs, scripts or music.
*All screenshots and videos are captured on PC build using HDPR(2020.3.5f1).

Support HDRP (a .unitypackage file included)
